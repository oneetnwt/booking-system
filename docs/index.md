## Overview

The K&A Resort Booking System is a comprehensive solution for managing resort bookings, featuring both guest-facing and administrative interfaces. The system handles everything from room availability checking to payment processing and invoice generation.

For quick reference, the main [README](../README.md) provides a high-level overview of the project, its features, and tech stack.

For support, please contact the development team or refer to the relevant documentation sections.
